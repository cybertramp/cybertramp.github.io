Build - nvme-cli

글을 쓴날: 2021-08-22

1. 클론/컴파일/설치

git clone https://github.com/linux-nvme/nvme-cli.git
cd nvme-cli
make
make install

# RUN
nvme --version

2. 커스텀 플러그인 추가

플러그인은 todd라는 명으로 추가한다.

mkdir -p plugins/todd
