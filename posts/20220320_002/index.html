Timer, Ticker

Timer는 1회 성으로 사용하는 것, Ticker는 주기적으로 실행하는 경우 사용하는 것이다.

ticker := time.NewTicker(time.Second * 2)

for currentTime := range ticker.C {
	fmt.Println("Tickered!")
}
