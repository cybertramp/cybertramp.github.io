  1. Windows 설정에서 앱 실행 별칭에서 Python 관련된 것을 모두 해제하고 하나 하나 올려보며 확인한다.

  2. 그래도 안되면 PyQT5를 그냥 재설치 해주면된다.

    pip install pyqt5
