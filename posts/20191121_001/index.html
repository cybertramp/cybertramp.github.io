Windows Terminal에서 시작 디렉토리 지정하기

글을 쓴날: 2019-11-23

Windows Terminal을 설치하고 실행하면 경로가 윈도우의 홈 디렉토리로 시작하게 된다. 작업하는데 매번 해당 경로를 타고 다시 이동해야하는 번거로움을 줄이기 위해 설정한다.

  • 기존 예제

    기존에는 "startingDirectory" 속성이 없어서 터미널 실행시 아래와 같은 위치로 시작한다.

    [설정된 속성]

    {
    "guid": "{c6eaf9f4-32a7-5fdc-b5cf-066e8a4b1e40}",
    "hidden": false,
    "name": "Ubuntu-18.04",
    "source": "Windows.Terminal.Wsl"
},

    [터미널 시작시]

    paran_son@pandorica:/mnt/c/Users/paran_son$

  • 위치를 WSL 내 사용자 홈 디렉토리로 변경

    여기서 home 디렉토리 아래 사용자명은 WSL 내 사용자 명으로 설정해주어야 한다.

    <수정된 속성>

    {
    "guid": "{c6eaf9f4-32a7-5fdc-b5cf-066e8a4b1e40}",
    "hidden": false,
    "name": "Ubuntu-18.04",
    "source": "Windows.Terminal.Wsl",
    "startingDirectory": "\\\\wsl$\\Ubuntu-18.04\\home\\paran_son"
},

    <터미널 시작시>

    greendot@pandorica:~$

  • 위치를 c 드라이브 내 특정 디렉토리로 변경

    여기서는 c드라이브 내 working 디렉토리를 시작 위치로 지정한다.

    <수정된 속성>

    {
	"guid": "{c6eaf9f4-32a7-5fdc-b5cf-066e8a4b1e40}",
    "hidden": false,
    "name": "Ubuntu-18.04",
    "source": "Windows.Terminal.Wsl",
	"startingDirectory": "../../working"
},

    <터미널 시작시>

    greendot@pandorica:/mnt/c/working$
