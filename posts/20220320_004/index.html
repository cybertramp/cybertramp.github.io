Powershell에서 Service control

글을 쓴날: 2022-01-03

Go언어로 Windows Service 프로그램을 만들면서 알게된 내용이다.

서비스 등록

sc.exe create "Go Windows Service" binPath= SERVICE_NAME\SERVICE_NAME.exe
sc.exe config SERVICE_NAME start=auto
sc.exe description "Go Windows Service" "Go Windows Service Description"
sc.exe start "Go Windows Service"

서비스 제거

sc.exe delete SERVICE_NAME

GoLang 관련

  • service로 사용시 fmt 라이브러리 사용하면 service 동작안함
  • windows 경로시 서비스로 돌기때문에 상대 경로로 넣으면 파일 못 불러옴
